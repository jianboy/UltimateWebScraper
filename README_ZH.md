# UltimateWebScraper

Ultimate Web Scraper 是一个非常棒的 chrome 插件，帮助快速提取网站数据，本项目基于此插件而来。

**去除导出限制，可轻松导出 csv，excel 等格式。**

## Usage


```
git clone https://github.com/jianboy/UltimateWebScraper.git

```
